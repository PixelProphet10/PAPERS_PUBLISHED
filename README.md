| Sr. No. | Paper Title | Detailed Description |

| **1** | **Review_on_Short-Circuit_Protection_Methods_for_SiC** | A comprehensive review of short-circuit protection techniques for Silicon Carbide (SiC) power devices. The paper covers fault detection mechanisms, active and passive protection circuits, real-time response strategies, and reliability enhancement for high-power converters using SiC MOSFETs/IGBTs. Includes comparison of methods, challenges, and future scope for high-speed switching environments. |
| **2** | **Fuzzy Logic Controller Based APF with Advanced Power Quality-2** | Presents the design and implementation of a **Fuzzy Logic Controller (FLC)** based Active Power Filter (APF) to mitigate harmonics and improve power quality under nonlinear load conditions. Includes system modeling, fuzzy rule base design, switching control, THD analysis, and MATLAB/Simulink verification. Demonstrates enhanced dynamic response compared to conventional control techniques. |
| **3** | **MULTILEVEL_13_LEVEL_H-BRIDGE_INVERTER** | Discusses the modeling and simulation of a **13-level cascaded H-bridge multilevel inverter**. Covers switching sequence development, pulse generation, output waveform synthesis, voltage-level formation, THD evaluation, harmonic spectrum analysis, and inverter performance under varying modulation indices. Highlights advantages in power quality and reduced switching stress. |
| **4** | **DC_MOTOR_CONTROL_USING_CHOPPER** | Focuses on the speed control of a DC motor using a **DC-DC chopper (PWM-based)**.
